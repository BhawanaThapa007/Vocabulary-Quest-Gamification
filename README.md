# 🎮 English Adventure - Interactive Vocabulary Learning App

An engaging and interactive vocabulary learning application designed for Grade 4-6 students. Master 80 advanced English words through 8 different activity types per unit!

![English Adventure](https://img.shields.io/badge/Grade-4--6-blue)
![React](https://img.shields.io/badge/React-18.2.0-61dafb)
![License](https://img.shields.io/badge/License-MIT-green)

## ✨ Features

### 🎓 Learning Content
- **4 Comprehensive Units**: Academic Excellence, Family Dynamics, Time Management, Emotions & Character
- **20 Words per Unit**: 80 total vocabulary words with definitions and context
- **8 Activity Types**: Picture Match, Word Match, Matching, Spelling, Listen & Click, Scramble, Fill Blanks, and Quiz

### 🎯 Gamification
- **Leaderboard System**: Track your progress against other learners
- **Achievement Badges**: Earn badges from Beginner 🌟 to Vocabulary Master 👑
- **Streak System**: Build streaks for bonus points
- **Hint System**: 5 hints available to help you learn
- **Star Rewards**: Collect stars for correct answers

### 📜 Certificates & Recognition
- **Downloadable Certificates**: Beautiful PNG certificates for each completed unit
- **Congratulation Messages**: Motivational messages upon completion
- **Progress Tracking**: Visual progress bars and statistics

### 🎨 User Experience
- **Unit Introductions**: Learn about each unit before starting
- **Personalized Welcome**: Random motivational messages
- **Progressive Unlocking**: Complete units to unlock the next
- **Responsive Design**: Works on desktop and mobile devices
- **Text-to-Speech**: Audio pronunciation for listening activities

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/english-adventure-app.git
cd english-adventure-app
```

2. **Install dependencies**
```bash
npm install
```

3. **Start the development server**
```bash
npm start
```

4. **Open your browser**
Navigate to `http://localhost:3000`

## 🎮 How to Use

1. **Enter Your Name**: Start by entering your name on the welcome screen
2. **Choose a Unit**: Select from 4 different vocabulary units
3. **Read Unit Info**: Learn about the unit's goals and content
4. **Complete Activities**: Work through 8 different activity types
5. **Use Hints**: Click the hint button if you need help
6. **Earn Rewards**: Collect stars, build streaks, and earn badges
7. **Download Certificate**: Get your certificate after completing each unit
8. **Track Progress**: Check the leaderboard and your stats

## 🎓 Educational Units

### Unit 1: Academic Excellence 🎓
Master essential academic vocabulary including: knowledge, assignment, curriculum, examination, comprehension, and more.

### Unit 2: Family Dynamics 👨‍👩‍👧‍👦
